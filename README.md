<h1><i>Змейка, написанная с помощью библиотеки arcade</i></h1>

Для запуска:

<code>git clone https://github.com/Danya05/Python_Snake.git</code><br>
<code>cd Python_Snake</code><br>
<code>git checkout dev</code><br>
<code>sudo apt install python3-pip</code><br>
<code>pip install arcade</code><br>
<code>python3 main.py</code><br>



<h2>Описание</h2>

<blockquote>Реализованна стандартная змейка, с возможностью перезапуска игры после проигрыша или выхода в главное меню.
<br>В главном меню можно увидеть змейку, которая ходит по кругу вокруг надписи и увеличивается со временем.
<br>Также реализована scoreboard система, каждая съеденная еда имеет стоимость - 100.

<br>При проигрыше показывается специальное окно, с которым можно взаимодействовать при помощи клавиатуры. Также при проигрыше змейка приобретает эффект мерцания.<img width="752" alt="Снимок экрана 2022-11-27 в 13 27 38" src="https://user-images.githubusercontent.com/112342386/204130421-51332553-0231-499b-bf93-438a2f73a596.png">
