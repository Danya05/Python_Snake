<h2>Описание</h2>

<blockquote>Реализованна стандартная змейка, с возможностью перезапуска игры после проигрыша или выхода в главное меню.
<br>В главном меню можно увидеть змейку, которая ходит по кругу вокруг надписи и увеличивается со временем.
<br>Также реализована scoreboard система, каждая съеденная еда имеет стоимость - 100.
